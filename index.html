Introduction to HTML (Hypertext Markup Language)

1. What is HTML?

HTML is a markup language used to describe the structure of a webpage. Unlike programming languages, HTML doesn't define behavior or logicâ€”it focuses on defining how content should be arranged and displayed in a web browser.

HTML uses tags to "mark up" text, images, and other content for display. Tags are used to wrap content and give it meaning (such as a header, a paragraph, or an image).

An HTML document is essentially a file (with the .html or .htm extension) that contains the content of a webpage, structured using HTML tags.

2. Basic Structure of an HTML Document

An HTML document follows a specific structure that helps browsers understand how to display the content. Here's the basic structure of an HTML document:

    
    
      
        
        
        My Webpage
      
      
        
Welcome to My Webpage

        
This is a paragraph of text on the webpage.

3. HTML Elements

HTML is composed of elements (also called tags). Elements are the building blocks of HTML and can be used to create content such as text, images, links, forms, and multimedia.

Most HTML elements are enclosed in opening and closing tags. For example, <h1> is the opening tag, and </h1> is the closing tag. The content goes between these tags.

<p>This is a paragraph.</p>

4. Common HTML Tags

Text Tags:

  • <h1> to <h6>: Headings of different levels (h1 is the largest, and h6 is the smallest).
  • <p>: Paragraphs of text.
  • <a>: Links to other pages or resources (with the href attribute).
  • <strong>: Bold text.
  • <em>: Italic text.

Image Tag:

<img src="logo.png" alt="Logo">

List Tags:

  • <ul>: Unordered (bulleted) list.
  • <ol>: Ordered (numbered) list.
  • <li>: List item.

Table Tags:

      <table>
        <tr>
          <th>Header 1</th>
          <th>Header 2</th>
        </tr>
        <tr>
          <td>Data 1</td>
          <td>Data 2</td>
        </tr>
      </table>

Form Tags:

      <form>
        <label for="username">Username:</label>
        <input type="text" id="username" name="username">
      </form>

5. HTML Attributes

HTML elements can have attributes that provide additional information about an element. Attributes are always specified in the opening tag.

      <a href="https://www.example.com">Visit Example</a>
      <img src="image.jpg" alt="An image description">

6. Nesting and Structure

HTML elements can be nested within other elements. This helps structure the content of the page logically.

      <div>
        <h2>Article Title</h2>
        <p>This is an article paragraph.</p>
      </div>

7. HTML5 Features

HTML5, the latest version of HTML, introduced several new features to improve web development, including:

  • Semantic Elements: Tags like <header>, <footer>, <article>, <section>, and <nav> make the page structure more meaningful and accessible.
  • Audio and Video: <audio> and <video> elements allow embedding media without relying on plugins like Flash.
  • Form Elements: HTML5 introduced new input types like email, date, and range for improved user interaction.
      <video controls>
        <source src="movie.mp4" type="video/mp4">
        Your browser does not support the video tag.
      </video>

Conclusion

HTML is the fundamental language for building web pages. It provides the structure and organizes content using tags. While HTML defines the content and structure, CSS is used for styling, and JavaScript adds interactivity. Understanding HTML is essential for anyone who wants to become a web developer, as it's the first step in creating anything on the web.
